# jbd-terrain-diffusion

## Overview

This repository is dedicated to processing Digital Elevation Models (DEMs), such as GeoTIFFs or similar raster formats, to generate a clean topographic surface.

While LiDAR-derived DEMs provide high-resolution terrain data, they often come with jagged, messy, and unreadable linework. The goal of this project is to apply advanced smoothing techniques directly to the elevation grid to create a high-quality, smoothed surface model that doesn't lose key topographical data.
